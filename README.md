EngWords Telegram Bot

Telegram-бот для изучения английских слов, разработанный на Kotlin. Проект демонстрирует интеграцию с Telegram Bot API, работу с базой данных SQLite через JDBC, обработку апдейтов и построение логики на основе пользовательских сессий.


Возможности

Добавление английских слов и переводов
Импорт слов из текстового файла (.txt)
Редактирование и удаление слов
Прикрепление изображений к словам через file_id
Режим обучения с вариантами ответа
Подсчёт прогресса изучения
Хранение данных в SQLite
Управление ботом через inline-кнопки


Технологии

Kotlin (JVM)
Coroutines
Telegram Bot API
kotlinx.serialization
SQLite через JDBC
Gradle Kotlin DSL
shadowJar


Архитектура проекта

api/ – работа с Telegram Bot API
bot/ – логика бота, обработка апдейтов, управление сессиями
database/ – слой работы с SQLite, SQL-запросы, репозиторий
utils/ – вспомогательные функции
serializableClasses/ – JSON-модели Telegram API
config/ – конфигурации и константы


Запуск

Установить JDK 21+

Установить переменную окружения BOT_TOKEN со значением токена бота

Выполнить сборку командой:
./gradlew shadowJar

Запустить приложение командой:
java -jar build/libs/EngWordsTelegramBot-all.jar

База данных Database.db создаётся автоматически при первом запуске.
